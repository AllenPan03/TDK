    TealUI

    开箱即用的 200 个组件，轻松打造高体验的中后台管理系统

    快速上手 为什么选择 TDK?

    安装

    							
							npm install tdk
						
    							
							yarn add tdk

    特性

    • 简单易用

      零配置快速上手。

    • 跨平台

      只需写一遍代码，即可在电脑、手机、平板中运行。

    • 扩展性强

      可以轻松扩展。

    • 组件完整

      无论是复杂的可拖拽的表格树，还是国内特有的城市选择、拼音模糊搜索，都能精确满足产品大大的变态需求。

    • 高性能

      支持百万级大数据、页面再多也不卡；框架底层的算法优化、无处不在的动画，确保流畅的用户体验。

    • 回归本质

      原生不依赖其它框架(但可混用)，不再折腾各种全家桶和打包配置；化繁为简，重新设计的完整前端解决方案，附赠全中文文档。
